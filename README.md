# React + Vite + TypeScript

Belajar _React + Vite + TypeScript_

```bash
npm create vite@4.1.0
```

React, TypeScript

```bash
npm install
```

```bash
npm install bootstrap@5.2.3
```

```bash
npm install styled-components
```

```bash
npm install immer@9.0.19
```

```bash
npm install react-hook-form@7.43
```
